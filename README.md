# Achtung Die Kugel

A fast local multiplayer snake/curve game on a sphere.

Steer left and right, avoid trails, and use gaps to survive.

This project is a tribute to [Achtung die Kugel](https://store.steampowered.com/app/938660/Achtung_die_Kugel/) and is intended as a direct copy of that game concept.
